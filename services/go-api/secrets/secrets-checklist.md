2. Annotate K8s ServiceAccount:
   ```yaml

     iam.gke.io/gcp-service-account: go-api@PROJECT_ID.iam.gserviceaccount.com
   ```
3. Remove GCP_* / GOOGLE_APPLICATION_CREDENTIALS from all configs
